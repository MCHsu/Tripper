## Technologies

- **[Framework]**
  - Vue 3 Composition API
  - Vue Router v4.x
  - Pinia
- **[Backend]**

  - Firebase v9

- **[Style]**
  - Tailwind CSS
  - RWD
  - SCSS
  
- **[Packages]**
  - Vuelidate
  - VueUse
  - Vue Toastification
  - VCalender
  - Swiper
  - Heroicons
